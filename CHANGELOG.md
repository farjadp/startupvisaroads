# Changelog

Record changes before each git push with date/time (minute precision).

## 2026-07-22 10:47
- **fix(ai): Upgrade Fal.ai model from flux/schnell (4 steps) to flux/dev (28 steps)** for significantly higher quality in-article photos.
- **feat(ai): Dual-type visual system (PHOTO / DIAGRAM)** — GPT-4o now decides whether each in-article visual should be a realistic photo (generated by Fal.ai) or a diagram/flowchart (generated directly as inline SVG by GPT-4o). SVG diagrams render crisp text, brand colors, and arrows — no more blurry/broken flowcharts.
- **fix(ai/generate): Critical Cloudflare 524 timeout fix** — duplicate detection was fetching full HTML content of all articles and running an O(n²) bigram comparison against the raw input, causing timeouts on large TEXT-mode inputs. Now only fetches article titles and truncates input to 400 chars, reducing complexity from billions to thousands of operations.

- **Base64 Image Storage**: Converted AI cover/in-text images and manual tiptap uploads to inline Base64 data URLs to fix 404 errors caused by stateless, multi-instance Cloud Run containers.
- **Blog System & Admin CRUD**: Implemented Farsi/English paginated blog index; created full article editor with Tiptap, cover image uploader, and instant draft/publish controls.
